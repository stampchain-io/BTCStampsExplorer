# Xverse Wallet Logo Assets

## Files

- `logo_xverse.svg` - Xverse "X" icon mark, 128x128, SVG format (primary asset)
- `logo_xverse.png` - Xverse icon, 128x128, PNG format (fallback)

## Source

- **SVG source**: Adapted from the Xverse Web Extension open-source repository
  - Repository: https://github.com/secretkeylabs/xverse-web-extension
  - Source file: `src/assets/img/createWalletSuccess/logo.svg`
  - Branch: `develop`
- **PNG source**: `src/assets/img/xverse_icon.png` from the same repository

## Brand Colors

- Background: `#181818` (dark)
- Icon: `#FFFFFF` (white)
- Accent: `#EE7A30` (Xverse orange)

## Usage

These assets follow the naming convention `logo_{provider}.{ext}` used across all wallet logos in this project.
The SVG is the preferred format as it scales perfectly at all sizes (24px to 128px).
